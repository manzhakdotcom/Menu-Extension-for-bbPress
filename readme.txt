== Installation ==

1. Upload the *"bbpress-menu-extension"* folder into the *"/wp-content/plugins/"* directory.
2. Activate the plugin through the *"Plugins"* menu in WordPress.
3. You can now add bbPress links in your Navigation Menus.
4. See FAQ for usage.
